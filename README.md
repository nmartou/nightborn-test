# nightborn-test project

Simple TO DO app

## How to start

Install the last version of the project

```
git clone https://github.com/nmartou/nightborn-test
cd ./nightborn-app
npm i
```

To start the project and open in `localhost:3000`, in the terminal write the following command :

```
npm run dev
```

To build the project 

```
npm run build
```

You can access to the main version [here](https://nightborn-test.vercel.app/)
